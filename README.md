### Pre-requisitos:
* ___python 3.9.0___: 
Lenguaje de progrmación que se utilizo para desarrollar el bot.
https://github.com/eternnoir/pyTelegramBotAPI
* ___telebot 3.7.5___: 
Una implementación de Python simple pero extensible para la API de Telegram Bot.
https://github.com/eternnoir/pyTelegramBotAPI
* ___API_TOKEN___: 
El API TOKEN de tu respectivo bot.
* ___pandas 1.2.0___: 
pandas es una herramienta de análisis y manipulación de datos de código abierto, rápida, potente, flexible y fácil de usar, construido sobre el lenguaje de programación Python.
https://pandas.pydata.org/
* ___openpyxl 3.0.6___: 
Una biblioteca de Python para leer / escribir archivos xlsx / xlsm de Excel 2010.
https://openpyxl.readthedocs.io/en/stable/
* ___tabulate 0.8.7___: 
Datos tabulares de impresión bonita en Python, una biblioteca y una utilidad de línea de comandos..
https://pypi.org/project/tabulate/


### Ejecución

Para la ejecución del bot, se deberá ejecutar el siguiente comando:

_Windows_
```console	
> python main.py
```

_Linux_
```console	
> ./python main.py

```
